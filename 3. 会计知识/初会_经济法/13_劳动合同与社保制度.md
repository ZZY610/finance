# 13_劳动合同与社保制度

## 📌 劳动合同与社保制度

### 一、劳动合同的订立

> **1. 订立原则**：合法、公平、平等自愿、协商一致、诚实信用<br>**2. 订立主体**：劳动者、用人单位<br>**3. 建立关系**：用工之日<br>**4. 非全日制用工**：可订立口头协议；可与多家订立但后订立不能影响先订立；不得约定试用期<br>**5. 合同效力**：<br>   &nbsp;&nbsp;(1) 劳动合同生效 ≠ 劳动关系建立<br>   &nbsp;&nbsp;(2) 无效合同，自始无效；部分无效，其余有效

---

### 二、劳动合同的必备条款

> **1. 合同期限**：固定期限、以完成一定工作任务为期限、无固定期限<br>**2. 工作时间**：每日8小时，每周40小时<br>**3. 报酬**：正常工资、加班工资、扣工资

---

### 三、劳动合同的可备条款

#### 1. 试用期

| 合同期限 | 试用期上限 |
| :--- | :--- |
| 非全日制用工、以完成一定工作任务为期限、不满三个月 | 不得约定试用期 |
| 3个月及以上不满1年 | ≤ 1个月 |
| 1年及以上不满3年 | ≤ 2个月 |
| 3年及以上固定期限、无固定期限 | ≤ 6个月 |

#### 2. 违反服务期约定是否支付违约金

> **(1) 用人单位提出解除劳动合同**：劳动者过错，劳动者支付违约金<br>**(2) 劳动者提出解除劳动合同**：用人单位无过错，劳动者支付违约金

---

### 四、劳动合同的解除和终止

#### 1. 经济补偿

```math
经济补偿金 = (本单位)工作年限 × 月(平均)工资

工作年限：
- 六个月以上不满一年算一年
- 六个月以下算半年

月工资：
1. = 劳动合同解除或终止前12个月的平均工资
2. < 当地最低工资标准，按当地最低工资标准计算
3. > 所在地区上年度平均工资3倍，按平均工资3倍的数额计算且补偿年限不超过12年
```

#### 2. 法律后果

> **1.** 双方劳动关系消灭<br>**2.** 合同归档保存至少两年<br>**3.** 用人单位出具离职证明并在15日内为劳动者转档<br>**4.** 用人单位违反规定解除或终止劳动合同，劳动者要求继续履行合同，用人单位应继续履行；劳动者不要求履行或劳动合同不能履行，用人单位应支付经济补偿标准2倍的赔偿金，支付了赔偿金则不再支付经济补偿

---

### 五、社会保险法律制度

#### 1. 基本养老保险

> **(1) 类型**：职工基本养老保险、城乡居民基本养老保险<br>**(2) 组成**：单位缴费、个人缴费、政府补贴<br>**(3) 待遇**：<br>   &nbsp;&nbsp;A. 基本养老金：按月领取<br>&nbsp;&nbsp;B. 丧葬补助金和遗属抚恤金：因病或非因工死亡，其遗属可以领取<br>&nbsp;&nbsp;C. 病残津贴：未达到法定退休年龄时因病或非因工致残完全丧失劳动能力的，可以领取

#### 2. 基本医疗保险

> **(1) 类型**：职工基本医疗保险、城乡居民医疗保险<br>**(2) 缴费**：<br>   &nbsp;&nbsp;A. 单位缴费：职工工资总额6%左右，单位缴费的30%转入个人账户<br>&nbsp;&nbsp;B. 个人缴费：本人工资收入的2%

#### 3. 工伤保险

> 职工不缴，单位缴

#### 4. 失业保险

> **(1) 缴费**：单位和个人的总和为1%，个人不超过单位<br>**(2) 领取期限**：<br>   &nbsp;&nbsp;A. 1年及以上5年以下 → 12个月<br>   &nbsp;&nbsp;B. 5年及以上10年以下 → 18个月<br>   &nbsp;&nbsp;C. 10年及以上 → 24个月<br>**(3) 标准**：自办理失业登记之日起算；最低生活保障标准 ≤ 失业保险金 ≤ 最低工资标准
