# 2_会计法律制度

## 📌 会计法律制度

### 一、会计工作管理体制考点

| 考点 | 要点 |
| :--- | :--- |
| **会计工作**<br>**管理体制** | **行政管理**（全国）：国务院财政部门<br>**行政管理**（本行政区域）：县级以上地方各级人民政府财政部门<br>**单位内部管理**：单位负责人（单位领导人）对会计工作、资料的真实性、完整性负责 |

---

### 二、会计档案管理考点

| 考点 | 要点 |
| :--- | :--- |
| **会计档案管理** | **保管期限**（永久）：年度报告 + 保管清册 + 销毁清册 + 鉴定书<br>**保管期限**（30年）：账 + 证 + 移交清册<br>**保管期限**（10年）：表 + 单 + 中期财务报告<br>**保管期限**（5年）：固定资产卡片（报废清理后） |

---

### 三、会计监督

| 会计监督类型 | 监督主体 | 监督对象 |
| :--- | :--- | :--- |
| **单位内部监督**（"内部"） | 会计机构、会计人员 | 本单位的经济活动 |
| **社会监督**（"外部"） | 1. 注册会计师及其所在的会计师事务所<br>2. 任何单位和个人（检举） | 1. 委托单位的经济活动<br>2. 违法行为 |
| **政府监督**（"行政机关"） | 财政、审计、税务、金融管理等部门 | 有关单位的会计资料 |
